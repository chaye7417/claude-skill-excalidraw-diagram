# Excalidraw 图表生成器 (Excalidraw Diagram)

将文本内容转换为 Excalidraw 图表，输出 Obsidian 兼容的 `.md` 文件。

## 功能

- 自动分析文本，识别概念和关系
- 生成流程图、思维导图、架构图等
- 输出 Obsidian 可直接打开的 Excalidraw 文件
- 支持多种图表类型和布局

## 使用方法

在 Claude Code 中说：
- "画一个流程图"
- "生成思维导图"
- "用 Excalidraw 可视化这个概念"

## 输出格式

生成的 `.md` 文件包含 Excalidraw 前置元数据，可在 Obsidian 中直接以 Excalidraw 视图打开和编辑。

## 依赖

- Obsidian
- Excalidraw 插件
